# snakes don't eat apples

![Cкриншот игры](snake_screenshot.png)


Моя первая игра на Pygame

## Запуск
1. Установите pygame: pip install pygame
2. Запустите snake.py

## Управление змейкой
Стрелки – направление
R – рестарт игры
Q – выход 

## Реализовано
· управление змейкой и ее рост
· рандомное появление мышей
· проигрыш при столкновениях
· использование спрайтов и поворот спрайта головы
· рестарт игры клавишей
